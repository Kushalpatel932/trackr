Q&A

What is Trackr?

Trackr is an open source project designed to help those struggling with addictions or other unwanted habits to break free of their addictions!

Is Trackr free?

Trackr is and always will be free! It is also open source.

Where is my data stored?

Trackr stores all data locally in the browser using Javascript. This means your data never leaves your PC. This means there is no need to log in or have an account, This keeps your data private and means our open source project doesn't need expensive servers to operate.

What does open source mean?

Open source is where the source code (what makes up the page you see and the code that runs the behind the scenes) is available for all to see. This means that developers can suggest improvement's and have them implemented making a better experience for you the user!

Why is Trackr open source?

Trackr is open to source to allow improvements from all the brilliant minds in web development community!
